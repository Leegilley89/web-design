There is quite a few table top rpg's out there. Pathfinder, starwars, dc, marvel, lor of the rings, you name it theres a table top game for it. dnd can bring people togetther. You can let your imagination run free and you can do almost anyhting if you can get lucky with the dice. i like dnd because i can take a break from normal life. i can be a completly out of this world and do things you couldn't normally do other wise.
